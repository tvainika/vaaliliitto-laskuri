# vaaliliitto-laskuri

Tämä laskuri auttaa arvioimaan teknisten vaaliliittojen hyötyä kuntavaalien tulosten perusteella.

Tiedoiksi on ladattu valmiiksi kuntavalien 2025 kuntakohtaiset tulokset.
